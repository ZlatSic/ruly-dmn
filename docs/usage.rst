Usage
=====

Installation
------------

To install ruly-dmn, call::

    pip install ruly-dmn

Command line
------------

Installing after installing ruly-dmn, the simplest way to use it is by creating
a DMN file in `Camunda Modeler`_ and calling the ``ruly-dmn`` command-line
program:

.. _Camunda Modeler: https://camunda.com/download/modeler/

.. program-output:: python -m ruly_dmn.main --help
